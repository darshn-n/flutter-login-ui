# Flutter Login UI

A new Flutter Login UI.


## Installation

You can clone by copy and pasting below commands in your Terminal.

```bash
    git clone https://github.com/darshn-n/flutter-login-ui/
    cd flutter-login-ui
    flutter run
```

App Folder Structure

